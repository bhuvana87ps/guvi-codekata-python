# GUVI CodeKata Python Solutions

This repository contains **200 Python logic-building problems** organized by topic.  
The problems are inspired by practice from the GUVI CodeKata platform and are structured as a **Python problem-solving roadmap**.

The goal of this repository is to strengthen **problem-solving skills, algorithmic thinking, and Python fundamentals**.

---

## Topics Covered

- Conditionals
- Loops
- Recursion
- Arrays
- Strings
- Mixed Logical Problems
